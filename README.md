# README

## Material Planner for Love & Deepspace

### 1. Need materials to upgrade a card

- Upgrading a card takes some time and a lot of material
- In steps of 10 levels this panel will show you how much material you need to upgrade a card to this level

### 2. Pulls needed

- This panel shows you how many saved pulls you need to guarantee myth pair or desired 5* card
- It already considers the free event pulls we get
- For reruns it is by default 140
- For Myths it is also 140 since you get 10 cumulative wish rewards on the way to the crate

### 3. Inventory

- Just a quick calculator that shows you how much your EXP Bottles are worth in total
